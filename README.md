# DateConverter

**DateConverter** — консольное приложение на C++ для генерации, валидации и анализа JSON-файлов с датами в различных форматах.

Программа позволяет:
- Генерировать корректные JSON-файлы с датами
- Валидация данных (форматы дат, корректность значений)
- Конвертировать даты между форматами: ISO 8601, Европейский, Американский
- Выполнять статистический анализ данных
- Обнаруживать ошибки в структуре JSON-файлов
- Измерять время выполнения операций (benchmarks)

## Запуск программы

1. Скачайте репозиторий и перейдите в папку `downloads/`
2. Запустите `dateconverter.exe`

## После запуска появится меню:
1. Генерация корректных JSON файлов
2. Генерация файлов с ошибками
3. Конвертация ISO-EDD.MM.YYYY
4. Конвертация ISO-MM/DD/YYYY
5. Анализ файлов
6. Запуск тестов
7. Бенчмарк производительности
8. Выход из программы

Корректных дат: 47О  
шибок: 3

Проверка корректности дат: 5.08.2023  
Результат: корректно
## Форматы дат

Программа работает с тремя форматами дат:
1. ISO 8601: 2024-12-11 (международный стандарт)
2. Европейский: 12.12.2024
3. Американский: 12/13/2024 (формат MM/DD/YYYY)
