# Проект: Конвертер дат (Date Converter) на C++

## Описание
Консольная утилита на C++ для нормализации и конвертации дат из различных текстовых форматов в ISO-формат и обратно в локальный формат. Поддерживает обработку массовых данных с отчётом о валидных и ошибочных записях.

## Функционал
- **Нормализация:** преобразование нестандартных форматов дат в ISO 8601 (`YYYY-MM-DD`)
- **Обратная конвертация:** из ISO в удобочитаемый русский длинный формат (например, `30 ноября 2025`)
- **Массовая обработка:** поддерживает пакетную конвертацию списка дат
- **Валидация:** проверка на корректность даты, учёт високосных лет, обработка неоднозначных форматов
- **Отчётность:** формирование двух таблиц:
  - Успешно конвертированные даты
  - Ошибки с указанием причины

## Технические требования
- **Надёжность:** корректный разбор неоднозначных форматов (например, `MM/DD/YYYY` vs `DD/MM/YYYY`)
- **Производительность:** оптимизация парсинга при массовой обработке, минимальное выделение памяти
- **Обработка ошибок:** устойчивость к пустым, битым или некорректным данным
- **Кросс-платформенность:** поддержка Linux, Windows, macOS

## Формат входных данных
Принимает JSON-массив объектов вида:

```json
[
  { "raw_date": "2025-11-30" },
  { "raw_date": "30 ноября 2025" },
  { "raw_date": "11/30/2025" }
]
Выходные данные
Таблица 1: Конвертированные даты
raw_date	iso_date	local_format
2025-11-30	2025-11-30	30 ноября 2025
30 ноября 2025	2025-11-30	30 ноября 2025
11/30/2025	2025-11-30	30 ноября 2025
Таблица 2: Ошибки обработки
raw_date	error_reason
31/02/2025	Некорректная дата
(пустая строка)	Пустая строка
invalid_date	Неизвестный формат даты
Использование
bash
# Компиляция
g++ -std=c++17 -o date_converter main.cpp date_parser.cpp json_handler.cpp -lstdc++fs

# Запуск
./date_converter --input dates.json --format ru_long
Поддерживаемые флаги:

--input: путь к JSON-файлу с датами

--format: целевой формат (iso или ru_long)

--output: путь для сохранения отчёта (опционально)

--help: показать справку

Пример работы
bash
$ ./date_converter --input sample.json --format ru_long
Вывод в консоль:

text
Конвертировано: 3
Ошибок: 1

Детальный отчёт сохранён в report_20250113.json
